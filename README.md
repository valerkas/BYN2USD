# BYN to USD Chrome Extension

Расширение для Chrome, которое находит на странице цены в рублях (BYN) и показывает рядом эквивалент в долларах США по курсу НБРБ.

## Что делает

- Получает официальный курс USD из API НБРБ.
- Кэширует курс в `chrome.storage.local` (обновление каждые 6 часов).
- Находит на странице цены вида `1234 руб`, `1 234 р.`, `999 BYN` и похожие.
- Добавляет долларовый эквивалент на следующей строке в формате `~$123.45`.

## Установка

1. Откройте `chrome://extensions`.
2. Включите **Режим разработчика**.
3. Нажмите **Загрузить распакованное расширение**.
4. Выберите папку проекта `e:\ChromePrices`.

## Пример

Было:

`Цена: 1 000 руб`

Стало:

`Цена: 1 000 руб`  
`~$311.52`  
(пример, зависит от текущего курса НБРБ)
